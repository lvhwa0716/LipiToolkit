Welcome to lipi-toolkit 4.0.1 Android
-------------------------------------

README for lipi-toolkit 4.0.1 Android

Package contents
----------------

The lipi-tookit4.0.1-android package comprises of the  following components:

1) source directory (src/) - This contains the source code of lipitk 4.0.0 for android.
2) binary file (lipitk.apk) - This is the sample application which can be executed on the android device.
3) documentation (doc/lipi-toolkit-android_4_0_user_manual.pdf) - This is the user manual to build lipitk source code for android using eclipse and also executing the sample application.


Extracting the package contents
-------------------------------

a. lipitk-toolkit4.0.1-android package : 

lipitk-toolkit4.0.1-android.zip - Binary/Source package for Windows Vista 

b. Unpacking/Extracting the package:

On Windows use winzip
	
	Double click and extract the contents from lipitk-toolkit4.0.1-android.zip file.
